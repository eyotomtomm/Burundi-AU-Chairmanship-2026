# Google Logo Asset Instructions

## Required File
**Filename:** `google_logo.png`
**Location:** `burundi_au_chairmanship/assets/icons/google_logo.png`

## Where to Get the Official Google Logo

### Option 1: Download from Google Brand Resources
1. Visit: https://developers.google.com/identity/branding-guidelines
2. Download the official Google "G" logo
3. Use the colored version (RGB: 66, 133, 244)
4. Recommended size: 48x48 dp or larger (96x96px @ 2x)

### Option 2: Use Material Icons (Temporary)
The code already has an error handler that will show a fallback icon if the image is missing, so the app will work without the logo file, just with a less polished appearance.

### Branding Guidelines
- Use official Google branding colors
- Don't modify or distort the logo
- Maintain proper spacing around the logo
- Follow Google's brand guidelines: https://developers.google.com/identity/branding-guidelines

## Current Status
The social authentication buttons are implemented and will work with or without this asset file. The missing asset will show a fallback icon until the proper logo is added.
