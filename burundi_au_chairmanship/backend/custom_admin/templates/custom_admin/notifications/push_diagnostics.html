help Troubleshooting Guide

Firebase fails
Set FIREBASE_CREDENTIALS_JSON in App Platform env vars with the service account key JSON from Firebase Console.
0 active tokens
Users must open the app and grant notification permission. The app registers its FCM token on startup.
Celery broker unreachable
Ensure REDIS_URL is set and the celery-worker component is running in App Platform.
Recipients = 0 in history
No tokens matched the targeting criteria. Check if the notification's targeting filters are too narrow.
Test push succeeds but nothing in system tray
Android: Settings → Apps → Be 4 Africa → Notifications → ensure "Default Notifications" channel is ON.
iOS: Settings → Be 4 Africa → Notifications → ensure Allow Notifications is ON. Also verify an APNs Key (.p8) is uploaded in Firebase Console → Project Settings → Cloud Messaging.
Test push fails with "Unregistered" error
The FCM token is stale (app was reinstalled or token expired). Open the app again to register a fresh token.
